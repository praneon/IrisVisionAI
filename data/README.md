# Data Folder

This repository keeps only lightweight, versioned data assets.

Tracked folders:
- `annotations/`: Pilot annotation notes and canonical CVAT export.
- `metadata/`: Canonical dataset metadata table (`metadata.csv`).
- `splits/`: Subject split manifests and split summary.

Not tracked in git:
- Raw image datasets
- Large derived artifacts
- Local scratch/intermediate data

Current annotation path:
- `data/annotations/pilot_v0_4/cvat_export/annotations.xml`
